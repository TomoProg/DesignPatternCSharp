# Memento

## そのパターンの用途、どんなケースに適用できるか
RedoやUndoなどを行いたい場合、あるときのスナップショットを取っておく必要がある。  
状態を保存しておいてその状態を復元したいというときに使える。  

## 作成したサンプルコードの概要説明
テキストエディタを題材にした。
- テキストを閲覧する機能
- テキストを編集する機能
- テキストを保存する機能
- テキストを前に保存した状態に戻す機能
を作成した。
