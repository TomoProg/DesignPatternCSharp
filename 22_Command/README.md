# Command

## そのパターンの用途、どんなケースに適用できるか
描画するとか、そういった仕事ごとにコマンドクラスに分ける。  
そしてそれを履歴として管理しておけば、再現もできるっていうのがこのCommandパターンの醍醐味だと思う。  

## 作成したサンプルコードの概要説明
炊飯器をコマンドパターンで表してみた。

以下の機能を持つ炊飯器を考える
- 炊飯: 設定した内容で炊飯を行う。炊飯が終わると設定を登録できる。  
- 炊飯モード設定: 早炊きなど炊飯モードの設定を行う。  
- 予約設定: 炊ける時間を設定する。  
- プリセットから設定: 登録した設定から炊飯器の設定を行う。
